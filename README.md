# IoT-Simulator
Test project to play with IoT and learn new things.

# Prerequisites
- [Docker](https://www.docker.com/)
- [Docker Compose](https://docs.docker.com/compose/)

## Execution
First run or after code changes:
``` ps
docker compose up -d --build
```

Subsequent runs:
``` ps
docker compose up -d
```

Stop the simulation:
``` ps
docker compose down
```
